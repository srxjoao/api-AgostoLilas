## ⚙️ Executar Projeto

```bash
# Clonar Projeto
$ git clone https://github.com/srxjoao/api-AgostoLilas.git

# Entrar na pasta do projeto
$ cd api-AgostoLilas

# Instalar as dependências
$ npm install

# Executar a aplicação
$ node index.js
```
---

## 🛠 Testando com Postman / Insomnia / ThunderClient

```bash
# ➤ Listar todos os centros de apoio
```
